## ✅ 핵심 게임루프

- [ ] **PLAY 버튼** → 런 시작 (로비 → 인게임 전환)
- [ ] **탭/클릭** → 광맥 HP 감소 + 피드백 표시
- [ ] **광맥 파괴** → 광석 드랍 + 다음 광맥 즉시 생성
- [ ] **콤보** → 연속 탭 시 콤보 카운터 증가, 1.5초 무탭 시 초기화
- [ ] **카드 오퍼** → 광맥 5개 파괴마다 카드 선택 화면 등장
- [ ] **카드 선택** → 런 모디파이어 즉시 반영
- [ ] **카드 리롤** → 크리스탈 소모 + 새 선택지 생성
- [ ] **깊이 자동 증가** → 광맥 10개 파괴마다 depth 증가 + 광맥 HP 상승
- [ ] **타임아웃** → 90초 후 자동 런 종료 + 결과 화면
- [ ] **Pause(나가기)** → 런 포기 + 로비 복귀

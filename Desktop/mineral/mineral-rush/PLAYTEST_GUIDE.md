## 참여해 주셔서 감사합니다!

Mineral Rush는 지하 광맥을 탭으로 채굴하는 2D 픽셀 아트 로그라이트 게임입니다.  
이번 플레이테스트는 **핵심 게임플레이 피드백**을 수집하기 위한 내부 테스트입니다.  
완성본이 아니기 때문에 그래픽·사운드는 아직 미완성 상태입니다.

---

## 🎮 플레이 방법

### 기본 조작
| 조작 | 행동 |
|---|---|
| **화면 탭 / 클릭** | 광맥 채굴 (곡괭이로 타격) |
| **연속 탭** | 콤보 게이지 유지 (1.5초 안에 탭해야 콤보 지속) |

### 게임 흐름
1. **PLAY** 버튼을 눌러 런 시작
2. 제한 시간(90초) 안에 광맥을 최대한 많이 부수세요
3. 광맥을 부수면 **광석**이 드랍되고, 일정 수 채굴 시 **카드 선택** 화면이 등장
4. 카드를 선택해 능력치를 강화하세요 (리롤 1회 가능, 크리스탈 소모)
5. 시간이 종료되면 결과 화면에서 보상(크리스탈)을 수령
6. 로비로 돌아와 **스킬트리**에서 영구 강화 가능

### 깊이(Depth) 시스템
- 더 깊은 층일수록 광맥이 강하고 보상도 많습니다
- 런 중 일정 광맥 수를 파괴하면 자동으로 깊이가 증가합니다

## 🔬 테스트 목표

이번 플레이테스트에서 특히 확인하고 싶은 항목입니다:

1. **타격감** — 탭할 때의 반응이 만족스러운가?
2. **카드 밸런스** — 카드 선택이 의미있게 느껴지는가? 명백하게 강하거나 약한 카드가 있는가?
3. **런 길이** — 90초가 너무 짧거나 길게 느껴지는가?
4. **콤보 시스템** — 콤보를 의식하며 플레이하는가? 콤보 창(1.5초)이 적절한가?
5. **스킬트리** — 크리스탈 소모가 적절한가? 어떤 노드를 먼저 올리게 되는가?
6. **진행 동기** — 다시 플레이하고 싶다는 욕구가 생기는가?

## 📋 세션 진행 방법

### 권장 플레이 시간
- 최소 **30분** (런 3~5회 이상)
- 가능하다면 **1시간** (스킬트리 업그레이드까지 체험)

### 권장 플레이 순서
1. 처음 2런은 PLAY 버튼(추천 깊이)으로 자연스럽게 플레이
2. 이후 스킬트리에서 노드 1~2개 업그레이드
3. 추가 2런 이상 플레이
4. 플레이테스트 설문 작성 (PLAYTEST_SURVEY.md)
5. 세션 종료 전 **텔레메트리 Export** 버튼 클릭해서 JSON 파일 저장

---

## 📤 플레이 데이터 제출 방법

1. 로비 화면 좌측 하단 **🔬 PLAYTEST PANEL** 에서 "텔레메트리 Export (JSON)" 클릭
2. 다운로드된 `mineral_rush_telemetry_XXXXXX.json` 파일을 개발팀에 전달
3. 설문 답변을 별도로 공유 (이메일 또는 슬랙)

> **개인정보**: 수집하는 데이터는 게임 내 행동(클릭 위치, 카드 선택, 런 결과)만 포함합니다.  
> 이름, 이메일, 위치 등 개인 식별 정보는 전혀 수집하지 않습니다.
